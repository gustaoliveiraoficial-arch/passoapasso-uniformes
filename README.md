# PassoaPasso — Vault Obsidian

## Estrutura

| Pasta | Uso |
|-------|-----|
| `Inbox/` | Notas rápidas, capturar ideias sem categorizar |
| `Diário/` | Notas diárias (geradas pelo Daily Notes) |
| `Projetos/` | Um arquivo por projeto ativo |
| `Pesquisas/` | Pesquisas e análises |
| `Templates/` | Templates reutilizáveis |
| `assets/` | Imagens e arquivos anexados |

## Templates disponíveis
- **Diário** — Para as notas do Daily Notes
- **Projeto** — Para novos projetos
- **Pesquisa** — Para pesquisas e análises
