## Objetivo

Criar scripts de abordagem personalizados para cada temperatura de lead, garantindo que a comunicação seja contextualizada, humana e com CTA claro para fechamento.

---

## Acceptance Criteria

- [ ] Script criado para lead ❄️ Frio (tom: retomada suave, sem pressão)
- [ ] Script criado para lead 🌡️ Morno (tom: acompanhamento, resolver dúvida)
- [ ] Script criado para lead 🔥 Quente (tom: urgência + condição especial)
- [ ] Script de follow-up (para quem não respondeu à 1ª mensagem)
- [ ] Cada script com no máximo 3 mensagens sequenciais
- [ ] Aprovado pelo responsável antes do envio em massa

---

## Scripts

### ❄️ Frio — Retomada Suave
```
Msg 1: "Oi [nome]! Tudo bem? A gente sabe que o tempo voa...
Você tinha pedido um orçamento do terceirão aqui com a gente.
Ainda tá nos planos? 😊"

Msg 2 (se não respondeu em 2 dias):
"[nome], só passando para dizer que a fila de produção de março
tá fechando. Se quiser garantir, é agora! Posso te mandar os detalhes?"

Msg 3 (último contato):
"Última mensagem, prometo! 😄 Se mudar de ideia sobre o uniforme
do terceirão, a gente tá aqui. Abraço!"
```

### 🌡️ Morno — Acompanhamento
```
Msg 1: "Oi [nome]! Você perguntou sobre o uniforme do terceirão
há alguns dias. Ficou alguma dúvida que posso esclarecer?
Tem alguma coisa que te travou na decisão?"

Msg 2 (se não respondeu em 1 dia):
"[nome], posso te contar um detalhe: grupos que fecharam
em fevereiro já estão em produção. Os de março entram na fila agora.
Quer garantir o seu?"
```

### 🔥 Quente — Urgência + Condição
```
Msg 1: "Oi [nome]! Você tava quase fechando o pedido do terceirão.
Não quero te perder! O que posso fazer para fechar hoje?
Tenho uma condição especial disponível até sexta."

Msg 2 (condição especial):
"Posso te oferecer [condição: prazo/brinde/parcelamento].
Mas é só até [data]. Topas? 🤝"
```
