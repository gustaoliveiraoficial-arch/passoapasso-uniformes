## 6. ESTRATÉGIA DE POSICIONAMENTO — PASSO A PASSO

### O posicionamento que está emergindo (e deve ser acelerado)

A Passo a Passo Uniformes está naturalmente construindo o posicionamento de:
> **"A empresa de uniforme que parece gente real"**

Isso se manifesta nos memes, nos bastidores, na linguagem próxima. **Esse é o diferencial** — a maioria dos concorrentes no nicho é fria e técnica.

### 7 passos para consolidar o posicionamento

**Passo 1: Humanização radical**
- Mostrar quem é a Jéssica (fundadora) com mais frequência
- Não apenas nos vídeos de storytelling, mas em posts casuais
- Meta: 1 aparição real por semana (não precisa ser vídeo com rosto)

**Passo 2: Cases de clientes como prioridade**
- Cada cliente que recebe uniforme = oportunidade de post
- Formato: foto recebimento + menção + "o que esse uniforme representa pra eles"
- Meta: 4-6 cases por mês (hoje: ~2/mês)

**Passo 3: Educação diferenciada (não genérica)**
- Parar de falar sobre tecido de forma técnica
- Falar sobre **o que o tecido faz pela equipe/empresa**
- Ex: não "DryFit tem secagem rápida", mas "Seu time vai parar de reclamar do uniforme pesado"

**Passo 4: Séries de conteúdo**
- Continuar e finalizar a série de história da empresa (ótimo início)
- Criar série "A turma que virou time" (formandos que receberam kit)
- Criar série "O uniforme por trás do negócio" (cliente real, história real)

**Passo 5: Sazonalidade planejada**
- Terceirão/Formandos: fev-mar ✅ (já fazendo)
- Times esportivos amadores: abr-mai (início das temporadas)
- Empresas novas: jan (início de ano/planejamento) — usar mais em jan/27
- Carnaval/eventos: jan ✅ (já explorado)

**Passo 6: Regularidade de formatos**
- Estabelecer 1 formato fixo por dia da semana
- Ex: Segunda = educação, Quarta = humor/meme, Quinta = case de cliente, Sexta = produto, Sábado = storytelling
- Isso treina o algoritmo E o público

**Passo 7: Engajamento ativo**
- Responder TODOS os comentários nos primeiros 60 minutos do post
- Essa é a janela que define se o algoritmo amplia o alcance ou não
- Fazer perguntas simples na legenda pra estimular comentários

## 8. DICAS PRÁTICAS IMEDIATAS

### O que fazer ESTA SEMANA

1. **Nunca poste 2 vezes no mesmo dia** — espaçar mínimo 24h entre posts
2. **Responda todos os comentários nas primeiras 2h** — o algoritmo mede velocidade de resposta
3. **Use 5-8 hashtags por post** (não 20+) — hashtags específicas de nicho > genéricas
4. **Hashtags recomendadas para o nicho:**
   - #uniformespersonalizados #kitformandos #terceirao2026
   - #passoapassouniformes #uniformeesportivo #personalizados
   - #camisetapersonalizada #moletomdaturma
5. **Sempre geotag na sua cidade** — aumenta alcance local
6. **Posta nos melhores horários para o seu nicho:**
   - Terça e Quarta: 19h-21h (melhor engajamento geral)
   - Sábado: 10h-12h (formandos online)
   - Domingo: evitar (menor alcance orgânico)

### O que parar de fazer

1. Parar de posts de "aviso operacional" (fechado pra manutenção, etc.) — use Stories para isso
2. Parar de CTA genérico "link na bio" sem contexto emocional antes
3. Evitar posts múltiplos no mesmo dia
4. Evitar posts só com texto e sem visual atrativo

## 9. COMO OBTER MÉTRICAS COMPLETAS DO INSTAGRAM

### Permissões necessárias

Para acessar as métricas reais de engajamento do Instagram via API, o token precisa ter:

| Permissão | Para que serve |
|-----------|---------------|
| `instagram_basic` | Dados básicos do perfil IG |
| `instagram_manage_insights` | Alcance, impressões, salvamentos por post |
| `pages_read_engagement` | Likes, comentários, compartilhamentos |

### Passo a passo para gerar o token correto

1. Acesse [developers.facebook.com](https://developers.facebook.com)
2. Vá em "Meus Aplicativos" → seu app
3. Em "Permissões e Recursos", solicite:
   - `instagram_basic`
   - `instagram_manage_insights`
   - `instagram_content_publish` (para postar via API)
4. Use o **Graph API Explorer** para gerar novo token com essas permissões
5. Certifique-se de selecionar a **Página** correta (Passo a Passo Uniformes)
6. O Instagram Business Account deve estar **vinculado à Página do Facebook**

### Verificar vinculação Instagram × Facebook

No Facebook:
- Acesse a Página → Configurações → Instagram → Conectar Conta

Com o token correto, é possível acessar:
- Alcance por post
- Impressões
- Salvamentos
- Compartilhamentos
- Cliques no perfil
- Dados de Stories (24h)
- Dados demográficos dos seguidores (gênero, idade, cidade)
- Horários de maior atividade do público
